# ShopEase - Mini Amazon Clone

A Java-based e-commerce application inspired by Amazon, developed using Java Swing and MySQL. The project demonstrates product browsing, shopping cart management, checkout, and user authentication.

## ✨ Features

- 🔐 User Login Authentication
- 🛍️ Product Catalog
- 🔎 Product Browsing
- 🛒 Shopping Cart
- 💳 Checkout System
- 📍 Address Entry
- 💰 Payment Method Selection
- 📦 Order Confirmation
- 🖼️ Product Images
- 🗄️ MySQL Database Integration
- ## 🛠️ Technologies Used

- Java
- Java Swing (GUI)
- JDBC
- MySQL
- IntelliJ IDEA
- Git & GitHub
- ## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/hacknivesh/ShopEase-mini-amazon-clone-.git
```

2. Open the project in IntelliJ IDEA.

3. Configure MySQL and import the required database.

4. Update the database credentials in:

```
src/com/ecommerce/util/DBConnection.java
```

5. Run:

```
Main.java
```

to start the application.
